# Leo 策略模板

`leo.yaml` 是从原社区完整 YAML 及其 OpenClash 覆写版本归一化得到的单一 Mihomo 模板。

## 使用

1. 把 `proxy-providers.Leo订阅.url` 替换为自己的 Clash/Mihomo 订阅地址。
2. 根据运行环境决定是否启用 `tun.enable`。
3. 在发布前确认远程 rule-provider 均可访问。

## 合并原则

- 扫描完整配置：59 份。
- 原始规则：2103 条；非 RULE-SET 内联规则归一去重后：146 条。
- 远程规则源按规范化 URL 精确去重后：720 个；同服务的不同规则源均保留。
- 同名但定义冲突的社区策略组没有机械拼接，而是映射到统一的地区、服务和兜底策略组。
- 广告类规则映射到 `REJECT`，国内和网络基础规则映射到 `DIRECT`，其余规则映射到对应服务组。
